## âš™ï¸ Getting Started (Local Development)

To run UltraNova locally, you will need to boot up both the UI and the API.

### 1. Start the API Engine
1. Navigate into the backend directory:
   ```bash
   cd ultranova-backend
   ```
2. Setup a virtual environment and install requirements:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```
3. Run the API:
   ```bash
   uvicorn app.main:app --reload
   ```

### 2. Start the Frontend OS
1. Open a new terminal and navigate into the frontend directory:
   ```bash
   cd ultranova-next
   ```
2. Install the necessary Node packages:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

**Visit `http://localhost:3000`** in your browser. Be sure to hit the physical Sun/Moon toggle switch in the navigation bar to experience the custom Canvas physics engines associated with both Dark and Light modes!
